# Client Manager Frontend

Frontend React della web app `Client Manager`.

## Stack

- React
- Vite
- React Router
- Axios
- Tailwind CSS

## Avvio

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## Variabili Ambiente

```env
VITE_API_URL=http://localhost:3000/api
```
